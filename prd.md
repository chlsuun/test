# 프로젝트 요구사항 정의서 (PRD)
## 프로젝트명: AI Gaming Agent Portfolio (가칭)

## 1. 프로젝트 개요
본 프로젝트는 Unity 엔진과 최신 LLM(Large Language Model) 기술을 결합하여, 게임 내에서 플레이어와 고도로 상호작용하는 "Intelligent Gaming Agent"의 가능성을 실증하기 위한 테크 데모 프로젝트입니다. 
세 가지 핵심 기능을 통해 심층적인 페르소나 구현, 외부 데이터 기반 경제 시뮬레이션, 그리고 논리적 추론 능력을 갖춘 NPC를 구현하는 것을 목표로 합니다.

## 2. 기획 의도 및 목표
- **Gaming Agent 역량 입증**: 단순 스크립트 기반이 아닌, LLM 기반의 능동적 상호작용이 가능한 NPC 개발 경험을 보여줍니다.
- **데이터 분석 및 활용**: 텍스트 마이닝, 감성 분석(Sentiment Analysis), RAG(검색 증강 생성) 등 다양한 데이터 처리 기술을 게임에 접목합니다.
- **정량적 성능 검증**: 프롬프트 엔지니어링 및 튜닝 전후의 성능 변화를 정량적으로 측정하여 공학적 접근 방식을 강조합니다.

## 3. 핵심 기능 (Key Features)

### 3.1. "Sayno의 가르침" 페르소나 NPC
- **기능 설명**: <세이노의 가르침> 데이터를 기반으로 구축된 강력한 페르소나를 가진 상점 주인 NPC.
- **기술적 특징**:
    - **Unity + LLM API 연동**: 실시간 대화 처리.
    - **Prompt Engineering**: "협상을 구걸로 착각하지 마라"와 같은 독설적이고 단호한 말투 및 가치관 주입.
- **사용자 경험 (UX)**: 플레이어가 가격 흥정을 시도할 때, 교훈적이면서도 냉철한 답변을 통해 단순 거래 NPC 이상의 몰입감을 제공.

### 3.2. LLM 기반 경제 뉴스 감성 분석 & 시장 예측 시스템
- **기능 설명**: 실제 경제 뉴스를 분석하여 게임 내 가상 자산(주식/코인)의 시세 변동을 결정하는 시스템.
- **기술적 특징**:
    - **Crawling & Sentiment Analysis**: 경제 뉴스 헤드라인 크롤링 및 LLM을 이용한 호재/악재 판별.
    - **Unity 연동**: 판별 결과를 게임 내 경제 수치에 실시간/주기적으로 반영.
- **사용자 경험 (UX)**: 현실 세계의 경제 흐름이 게임 내 시장에 영향을 미치는 "현실 연동형 경제 시스템" 체험.

### 3.3. "추리 게임" 거짓말 탐지기 에이전트 (RAG 활용)
- **기능 설명**: 범인의 알리바이와 증거 데이터를 토대로 플레이어와 두뇌 싸움을 벌이는 범인 NPC.
- **기술적 특징**:
    - **RAG (Retrieval-Augmented Generation)**: 제한된 증거 데이터(JSON) 내에서 답변을 생성하도록 제어.
    - **Deception Prompting**: 교묘하게 거짓말을 섞어 답변하도록 튜닝.
    - **성능 지표화**: 프롬프트 개선에 따른 거짓말 성공률 변화(예: 60% -> 85%)를 측정 및 리포트화.
- **사용자 경험 (UX)**: 논리적 허점을 파고들어야 하는 고난도 추리 게임 플레이.

## 4. 기술 스택 (Tech Stack)
- **Game Engine**: Unity (C#)
- **AI/ML**: OpenAI API (GPT-4/3.5) or Local LLM, LangChain (Optional for RAG)
- **Data**: Python (Crawling, Pre-processing), JSON (Game Data)

## 5. 개발 로드맵 및 검증 계획
- **Phase 1: Sayno NPC**: 기본 대화 시스템 구축 및 페르소나 프롬프트 튜닝. (검증: 대화 로그 분석)
- **Phase 2: Economy System**: 뉴스 크롤러 구현 및 유니티 연동 테스트. (검증: 뉴스 감성 점수와 게임 내 가격 변동 상관관계 확인)
- **Phase 3: Detective Agent**: RAG 파이프라인 구축 및 거짓말 시나리오 테스트. (검증: 벤치마크 셋을 통한 거짓말 성공률 측정)

## 6. 기대 효과 (Deliverables)
- 실행 가능한 Unity 프로젝트 빌드
- 프롬프트 엔지니어링 실험 보고서 및 성능 지표 데이터
- 시연 영상 (포트폴리오용)
