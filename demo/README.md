# 세이노의 상점 - 웹 데모

브라우저에서 바로 체험 가능한 세이노 NPC 채팅 데모입니다.

## 🎮 Live Demo
**지금 바로 체험하기**: [https://chlsuun.github.io/test/demo/](https://chlsuun.github.io/test/demo/)

---

## 💡 Features

### 🗣️ AI 대화
- **Mock AI**: API 없이 작동하는 키워드 기반 응답
- **페르소나**: 독설적이지만 현명한 세이노의 성격 반영
- **자연스러운 대화**: 다양한 키워드 인식

### 🛒 상점 시스템
- **8가지 아이템**: 검, 갑옷, 포션 등
- **구매/판매**: 실시간 거래 가능
- **인벤토리**: 구매한 아이템 관리
- **동적 가격**: 판매가는 구매가의 60%

### 🎨 UI/UX
- **다크 테마**: 모던한 그라데이션 디자인
- **반응형**: 모바일/태블릿/데스크톱 지원
- **실시간 업데이트**: 소지금, 인벤토리 즉시 반영
- **애니메이션**: 부드러운 메시지 표시

---

## 🎯 사용 방법

### 기본 명령어
- `목록` - 상품 목록 보기
- `인벤토리` - 내 소지품 확인
- `강철검 주세요` - 아이템 구매
- `강철검 팔게요` - 아이템 판매
- `안녕하세요` - 인사

### 빠른 시작
1. 링크 접속: https://chlsuun.github.io/test/demo/
2. 하단의 버튼을 클릭하거나 직접 메시지 입력
3. 세이노와 대화하며 상점 체험!

## 🛠️ 기술 스택

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **AI**: Mock AI (키워드 기반 응답 시스템)
- **Deployment**: GitHub Pages
- **No Backend**: 완전한 클라이언트 사이드 앱
