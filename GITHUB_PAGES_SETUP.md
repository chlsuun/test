## 🚀 GitHub Pages 활성화 (필수!)

### 방법 1: GitHub 웹사이트에서 설정

1. **GitHub 저장소로 이동**
   ```
   https://github.com/chlsuun/test
   ```

2. **Settings 클릭**
   - 저장소 상단 메뉴에서 "Settings" 클릭

3. **Pages 메뉴 선택**
   - 왼쪽 사이드바에서 "Pages" 클릭

4. **Source 설정**
   - Branch: `main` 선택
   - Folder: `/ (root)` 선택
   - **Save** 클릭

5. **완료!**
   - 몇 분 후 다음 주소에서 확인 가능:
   ```
   https://chlsuun.github.io/test/
   ```
